Important Concepts

The World Wide Web

The web is a bunch of computers that communicate with each other. When a person goes to a web page like www.google.com, their computer sends a HTTP (Hypertext Transfer Protocol) Request to a server. The server finds the appropriate HTML (Hypertext Markup Language) document and sends it back to the user's computer where a web browser (for example Chrome, Safari, Firefox, Internet Explorer or Brave) interprets the page and displays it on the user's screen. For this reason, HTML is the main type of document on the web.

Components of the Web

  • HTTP: the main protocol of the web
  • Servers: computers that host the files that make up the web
  • Internet: the world's largest computer network
  • Browser: a program that runs on your computer to display the files found on the web

Basic HTML Elements

HTML elements are what we use to tell a web browser how to display content on a web page.
Most HTML elements consist of an opening and a closing tag with some content between them.

  • <b>text</b> Makes the 'text' content bold
  • <em>text</em> Makes the 'text' content italic

HTML Attributes

Some elements accept additional values called attributes in their opening tags. Attributes provide additional information to the browser.

  • <a href="url">link</a> Creates a hyperlink to the href attribute value url and names it
  • <img src="url" alt="text"> Displays the image hosted at the scr attribute value "url". If the image cannot be displayed, then the alt attribute value "text" is shown.

More HTML Elements

  • <p>text</p> Defines a paragraph. The content of the <p> elements is in a "box"
  • <br> Creates a line break
  • <div>content</div> Creates a "box" container element
  • <span>content</span> Creates an "inline container element
