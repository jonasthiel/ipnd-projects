Lesson 1: The Basics of the Web and HTML

The World Wide Web

The web is a bunch of computers that communicate with each other. When a person goes to a web page like www.google.com, their computer sends a HTTP (Hypertext Transfer Protocol) Request to a server.

The server finds the appropriate HTML (Hypertext Markup Language) document and sends it back to the user's computer where a web browser (for example Chrome, Safari, Firefox, Internet Explorer or Brave) interprets the page and displays it on the user's screen. For this reason, HTML is the main type of document on the web.

Components of the Web

HTTP: the main protocol of the web

Servers: computers that host the files that make up the web

Internet: the world's largest computer network

Browser: a program that runs on your computer to display the files found on the web

Basic HTML Elements

HTML elements are what we use to tell a web browser how to display content on a web page.

Most HTML elements consist of an opening and a closing tag with some content between them.

<b>text</b> Makes the 'text' content bold

<em>text</em> Makes the 'text' content italic

HTML Attributes

Some elements accept additional values called attributes in their opening tags.

Attributes provide additional information to the browser.

<a href="url">link</a> Creates a hyperlink to the href attribute value url and names it link

<img src="url" alt="text"> Displays the image hosted at the scr attribute value "url". If the image cannot be displayed, then the alt attribute value "text" is shown.

Lesson 2: Creating a Structured Document with HTML

Developer Tools

Every modern web browser includes a powerful suite of developer tools. These tools do a range of things, from inspecting currently-loaded HTML, CSS and JavaScript to showing which assets the page has requested and how long they took to load.

For exmaple, to open the developer tools on Chrome (on Mac), use the keyboard shortcut Ctrl Option I. Alternatively, you can use the Chrome menu in the browser window, select the option "More Tools", and then select "Developer Tools".

HTML-CSS-DOM

HTML (Hypertext Markup Language) the standard markup language used to create web pages.

CSS (Cascading Style Sheets) style sheet language used for describing the look and formatting of a document written in a markup language.

DOM (Document Object Model) a cross-platform and language-independent convention for representing and interacting with objects in HTML (and other markup languages). The nodes of every document are organized in a tree structure, called the DOM tree.

Boxes

Everything on a webpage is a box. Breaking up sections of the page with div tags allows us to neatly boxify the page.

All boxes are made up of 4 components: the margin, border, padding, and content.

Lesson 3: Adding CSS Style to HTML Structure

CSS

Cascading Style Sheets follow the Most Specific Rule Applied: you go down selectors until you reach the most relevant one.

A selector referencing either an HTML tag, .class, or #id. Then define properties and values to the selector.

Reference the CSS stylesheet in the <head> of the HTML document: <link rel="stylesheet" href="style.css">
